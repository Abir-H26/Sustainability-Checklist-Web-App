##  Context

Developed at the **ClimateAction.tech Hackathon** in Zurich — a competitive event bringing together developers, designers, and sustainability professionals to prototype solutions addressing real environmental challenges.

Our team identified a gap: IT projects lack structured guidance on environmental impact. We built a GenAI-powered checklist that meets teams wherever they are in their project lifecycle.

**Key highlights:**
- Prototyped a Gemini-powered web app generating sustainability recommendations across IT project phases
- Presented the solution live to the full hackathon audience — industry professionals and mentors
- Collaborated across a multidisciplinary team to deliver a working MVP under time pressure
- Prototype attracted investor interest from industry mentors and industrial companies

##  How to Run Locally

1. Clone the repo:
```bash
   git clone https://github.com/Abir-H26/Sustainability-Checklist-Web-App.git
```
2. Open the project folder and double-click `index.html` — it opens in your browser. No server needed.

---

##  How to Deploy (GitHub Pages)

1. Make sure `index.html` and `qrcode.png` are at the root of the `main` branch
2. Go to **Settings → Pages** in this repo
3. Under **Source**, select branch `main` and folder `/ (root)`
4. Click **Save**
5. After ~1 minute your page is live at:
   `https://abir-h26.github.io/Sustainability-Checklist-Web-App/`
